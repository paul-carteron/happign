# List of all API keys from IGN

All API keys are manually extract from this
[table](https://geoservices.ign.fr/documentation/services/tableau_ressources)
provided by IGN.

## Usage

``` r
get_apikeys()
```

## Value

`character`

## Examples

``` r
if (FALSE) { # \dontrun{
# One API key
get_apikeys()[1]

# All API keys
get_apikeys()
} # }
```
