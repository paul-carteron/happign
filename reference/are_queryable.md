# are_queryable

Check if a wms layer is queryable with GetFeatureInfo.

## Usage

``` r
are_queryable(apikey)
```

## Arguments

- apikey:

  API key from
  [`get_apikeys()`](https://paul-carteron.github.io/happign/reference/get_apikeys.md)
  or directly from the [IGN
  website](https://geoservices.ign.fr/services-web-experts)

## Value

`character` containing the name of the queryable layers
